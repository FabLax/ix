## Breaking changes for designers

### Deprecating Figma libraries

The following Figma libraries are replaced by the _iX Component_ library:

- iX Component - Brand Light
- iX Foundation - Brand Light
- iX Foundation - Brand Dark

Please follow the instructions below to update your Figma files to the new iX Component library:

<table className="table-no-border">
  <tbody>
    <tr>
      <td style={{ verticalAlign: 'top' }}>
          1. On the left panel, click on **Assets**.
          2. Click on the <ix-icon name="book" size="16"></ix-icon> icon to open the libraries dialog.
          3. Select a library you want to replace.
          4. Click **Swap library**.
          5. Choose **iX Components**.
          6. Confirm and repeat for other libraries.
          <br/>
          > Note: Video can only be accessed **Siemens internally**.
      </td>
      <td style={{ verticalAlign: 'top', paddingLeft: '2rem' }}>
        <iframe src="https://siemens.sharepoint.com/teams/DICTOTOUX/_layouts/15/embed.aspx?UniqueId=cb359e96-bda4-4167-bc16-948343509d33&embed=%7B%22ust%22%3Atrue%2C%22hv%22%3A%22CopyEmbedCode%22%7D&referrer=StreamWebApp&referrerScenario=EmbedDialog.Create" width="640" height="360" frameborder="0" scrolling="no" allowfullscreen title="Figma tutorial - swap library.mp4"></iframe>      </td>
    </tr>
